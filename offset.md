# offset : 선택한 요소의 좌표를 가져오거나 특정 좌표로 이동

   
    ■ var abc = $( 'h1' ).offset();
    
    ---> h1 요소의 좌표를 변수 abc에 저장

    ■ $( 'h1' ).offset( { left: 100, top: 50 } );

    ---> h1 요소를 왼쪽 100ox, 위 50px 만큼 위치이동
